# Branding

Replace `public/branding/tomduko-logo.svg` and `public/icons/master.svg`. Keep their paths stable, then run `npm run icons:generate`. Confirm `icon-180.png`, `icon-192.png`, `icon-512.png` and `icon-maskable-512.png`, rebuild, inspect on both light/dark iPhone wallpapers and reinstall the PWA to refresh a cached Home Screen icon.
