# Physical iPhone QA

Browser emulation cannot validate all iOS behaviour. On intended hardware verify: Safari installation and icon; standalone launch; notch/Dynamic Island and Home-indicator clearance at 375, 390 and 430 CSS pixels; no bounce, selection, callout or zoom; rapid number entry; keyboard is never summoned; background/lock timer behaviour; exact close/relaunch restoration; Airplane Mode relaunch; reduced motion, VoiceOver labels, contrast and hardware orientation changes; and update acceptance during an unfinished puzzle.
